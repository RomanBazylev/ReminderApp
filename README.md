# 🎉 Напоминалка — Поздравления & Задачи

Бесплатное веб-приложение для управления поздравлениями, контактами и периодическими email-рассылками. Работает как статический сайт (GitHub Pages), все данные хранятся в localStorage.

## Возможности

### 📋 Поздравления
- **Контакты** с категориями (Семья, Друзья, Коллеги, Крестники)
- **Праздники** — встроенные российские + свои
- **Шаблоны поздравлений** с плейсхолдерами `{name}`, `{sender}`, `{age}`, `{holiday}`
- **12 каналов отправки**: WhatsApp, Telegram, Viber, Email, SMS, VK, ОК, Facebook, Instagram, Discord, Telegram Bot, копирование
- **Авто-отправка** через Telegram Bot API (100% бесплатно)

### 🔄 Периодические задачи
- Регулярные email-запросы по расписанию (еженедельно / раз в 2 недели / ежемесячно)
- **Ротация текстов** — каждый раз отправляется следующий вариант, чтобы не выглядеть как спам
- Встроенный шаблон: запрос мест в садик (Польша, 6 вариантов на польском)
- Открывает `mailto:` ссылку с предзаполненными полями
- Журнал отправок

### ⚙️ Прочее
- Тёмная / светлая тема
- Экспорт / импорт данных (JSON)
- Браузерные уведомления
- Миграция данных из v1

## Запуск

### GitHub Pages (рекомендуется)
1. Пушните репозиторий на GitHub
2. Settings → Pages → Source: **main** branch, folder: **/ (root)**
3. Сайт доступен по адресу `https://<username>.github.io/ReminderApp/`

### Локально
Просто откройте `index.html` в браузере — всё работает без сервера.

## Технологии
- Один HTML-файл, vanilla JavaScript, CSS Variables
- localStorage для хранения данных
- Telegram Bot API для авто-отправки
- `mailto:` deep links для email-задач
- Никаких зависимостей, сборки или серверной части
